   • 学习市场分析
   • 开发投资工具
   • 非高频交易

✅ 优势:
   • 完全免费
   • 无需账号
   • 简单易用
   • 数据准确
   • 实时更新(秒级)

   stock = eastmoney_service.get_stock_realtime('000001')


【进阶：EMT API】(需要账号)

📊 适用场景:
   • 专业量化交易
   • 高频交易策略
   • 实盘程序化交易
   • 毫秒级数据需求

📊 优势:
   • 毫秒级实时
   • Level-2行情
   • 支持交易功能
   • 专业级数据

⚠️  使用前提:
   1. 开通东方财富证券账号
   2. 申请EMT API权限
   3. 获取服务器地址
   4. 完成C++封装或使用官方SDK

⚠️ 重要提示
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. EMT API 需要账号
   ✅ 如果没有账号，继续使用 HTTP API 即可
   ✅ HTTP API 完全够用，数据准确实时

2. EMT API 是 C++ 接口
   ⚠️  需要完整的 C++ 封装才能在 Python 中使用
   ⚠️  建议联系东方财富获取官方 Python SDK

3. 交易功能需谨慎
   ⚠️  实盘交易风险极高
   ⚠️  需要额外的审批和权限
   ⚠️  建议专业人士使用

4. 数据安全
   🔐 不要将账号密码写在代码里
   🔐 使用环境变量或配置文件
   🔐 配置文件添加到 .gitignore
